# Sistema de Gestión de Inventario

## Descripción del sistema
Esta es una aplicación de consola desarrollada en Python diseñada para administrar los productos de una tienda pequeña. Permite registrar nuevos productos, listar el inventario actual, actualizar cantidades, eliminar artículos y calcular el valor total del stock.

## Tecnologías utilizadas
* **Lenguaje:** Python 3
* **Control de Versiones:** Git y GitHub
* **Estándar de Commits:** Conventional Commits


---

## Instrucciones de ejecución
1. Asegúrate de tener instalado Python 3.
2. Abre una terminal en la carpeta del proyecto.
3. Ejecuta el siguiente comando:

   ```bash
   python3 main.py
   ```
